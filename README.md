# AI-Powered Email Client

## Overview
A full-stack AI-powered email client focused on intelligent drafting, thread summarization, and efficient email retrieval.  
Designed to explore scalable SaaS architecture, AI integration, and production-grade authentication and payments.

## Key Features
- Context-aware AI email drafting and summarization
- Thread-level search and retrieval
- Subscription-based premium AI features
- Secure authentication and user management
- Scalable database and storage integrations

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI API
- Prisma ORM
- PostgreSQL
- Stripe (subscriptions + webhooks)
- Clerk (authentication)
- AWS S3
- Pinecone (vector search)

## Architecture Notes
- Modular API and service layers for maintainability
- Webhook-based billing event handling
- Vector search for semantic email retrieval
- Designed with scalability and reliability in mind
